## Install on Mac
1. Open Terminal.app in ```GGB-module-main``` folder

2. Write following commands:
```
pip3 install keyboard
pip3 install pynput
pip3 install pyperclip
sudo python3 wrap.py
```

3. Allow access to Terminal in System Preferences
